# Smart Shopping Cart System

A simple desktop shopping cart application made with Python and PyQt5. 

## Features
* **Manage Items:** You can add or remove items easily.
* **Discount System:** It automatically calculates discounts and the total price. You can also use coupon codes like `WELCOME10`.
* **Live Summary:** It shows your current cart summary and total cost in real-time.
* **Clean UI:** A basic and easy-to-use graphical interface.

## How to Run
1. Install the required library: `pip install PyQt5`
2. Run the application: `python main.py`
